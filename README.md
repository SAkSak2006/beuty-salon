# 🌸 BeautySalon - Информационная система салона красоты

Современная, полнофункциональная информационная система для управления салоном красоты с интуитивно понятным интерфейсом и богатым функционалом.

## 📦 Установка и запуск

### Требования
- Современный веб-браузер (Chrome, Firefox, Safari, Edge)
- Веб-сервер (опционально, можно открывать напрямую)

### Быстрый старт

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/SAkSak2006/beauty-salon.git
   cd beauty-salon
   ```

2. **Откройте файл index.html**
   - Просто откройте `index.html` в браузере, или
   - Используйте локальный сервер:
   ```bash
   # С помощью Python
   python -m http.server 8000

   # С помощью Node.js
   npx http-server

   # С помощью PHP
   php -S localhost:8000
   ```

3. **Начните работу!**
   - Система автоматически загрузит демо-данные при первом запуске
   - Все данные хранятся локально в вашем браузере

## 📖 Руководство пользователя

### Первый запуск

При первом запуске появится экран приветствия с кратким туром по системе. Вы можете:
- Начать работу с демо-данными (рекомендуется)
- Начать с пустой базы данных
- Импортировать существующие данные

### Горячие клавиши

| Комбинация | Действие |
|-----------|----------|
| `Ctrl + K` | Глобальный поиск |
| `Ctrl + N` | Новая запись (на соответствующей странице) |
| `Ctrl + H` | История действий |
| `Ctrl + ,` | Настройки |
| `Ctrl + /` или `?` | Справка по горячим клавишам |
| `Ctrl + Z` | Отменить действие |
| `Ctrl + Y` | Повторить действие |
| `Esc` | Закрыть модальное окно |

### Навигация

Используйте верхнее меню для переключения между разделами:
- 🏠 **Главная** - дашборд и статистика
- 📅 **Записи** - управление записями
- 👥 **Клиенты** - база клиентов
- 👔 **Сотрудники** - управление персоналом
- ✂️ **Услуги** - каталог услуг
- 📦 **Склад** - складской учет
- 📊 **Отчеты** - аналитика и отчеты

### Работа с данными

#### Создание новой записи
1. Перейдите в раздел **Записи**
2. Нажмите кнопку **Новая запись** или `Ctrl + N`
3. Заполните форму
4. Нажмите **Сохранить**

#### Добавление клиента
1. Перейдите в раздел **Клиенты**
2. Нажмите **Добавить клиента**
3. Введите контактные данные
4. Сохраните

#### Складские операции
1. Перейдите в раздел **Склад**
2. Используйте вкладки:
   - **Остатки на складе** - текущие запасы
   - **Движение товаров** - история операций
   - **Закупки** - заказы поставщикам
   - **Аналитика** - ABC-анализ, оборачиваемость
   - **Настройки** - привязка материалов к услугам

### Экспорт/импорт данных

#### Экспорт базы данных
1. Откройте настройки (`Ctrl + ,`)
2. Нажмите "Экспорт/Импорт"
3. Выберите "Экспортировать базу данных"
4. Файл будет сохранен в формате JSON

#### Импорт базы данных
1. Откройте настройки
2. Нажмите "Экспорт/Импорт"
3. Выберите "Импортировать базу данных"
4. Выберите ранее сохраненный JSON файл
5. Подтвердите замену данных

## 🔧 Настройка

### Изменение цветовой схемы

В `index.html` найдите секцию Tailwind Config:

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                primary: '#ec4899',    // Розовый
                secondary: '#8b5cf6',  // Фиолетовый
                accent: '#f472b6',     // Светло-розовый
            }
        }
    }
}
```

### Настройка хранилища

По умолчанию используется LocalStorage. Для изменения лимитов или типа хранилища отредактируйте `js/database.js`.

### Кастомизация функционала

Все компоненты модульные и легко настраиваются. Найдите нужный компонент в `js/components/` и внесите изменения.

## 🐛 Решение проблем

### Данные не сохраняются
- Проверьте, что LocalStorage не заблокирован в браузере
- Очистите кеш браузера
- Проверьте квоту LocalStorage (обычно 5-10MB)

### Медленная работа
- Очистите старые данные
- Уменьшите количество записей в истории
- Проверьте расширения браузера

### Не отображаются графики
- Убедитесь, что загружена библиотека Chart.js
- Проверьте консоль браузера на ошибки
- Обновите страницу
